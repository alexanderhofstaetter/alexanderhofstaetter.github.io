        create_task(requester_id, opts)

        @spec create_task(String.t(), [task_option()]) ::
  {:ok, Owat.Model.Task.t()}
  | {:error,
     :not_found
     | :no_subject_encounter_match
     | :no_description
     | Owat.CrudUtils.validation_error()}

        Create a task on an encounter.

        The status will be set to requested and the requester will be set to the user making the request. The sort_order will be set according to the before value if it matches another task_id on the encounter, or to the last position in the encounter. If code is provided, it has to match an existing task template. patient_instruction cannot be provided without setting patient_visible to true.
