        Owat.Gin.Signing (PVE v0.0.1)

        Signing supervisor

        This module handles GINO-signing related functionality. It is meant to be started in the supervision tree and supervises the Signing-Jobs.

        ready?()

        @spec ready?() :: boolean()

        Returns true if the system should generally be ready to handle signatures.

        This tests that all these conditions are met:

        1. There is a process registered to the module atom
        2. The process is alive
        3. There are readers available
        4. At least one of these readers is marked as admin
        5. There is at least one admin card present on an admin card reader
