        Owat.Medication (PVE v0.0.1)

        Functions relating to prescriptions.

        Workflow of prescribing medications

        A prescription may or may not start with a proposal request. A proposal request can be accepted by a practitioner to become a draft order. The draft order exists because a prescription may prescribe multiple products at once. Issuing the prescription will set all draft orders to active and set the group identifier.

        Accepts a request for a medication.

        Sets the medication request to order intent. This means a prescription is about to be issued but since it may be composed of multiple requests this function does not issue it yet.

        One may optionally supply a dosage when accepting a request.

        Accepting a request requires the request to be in status active with intent proposal, otherwise the error :invalid_status will be returned.
