        Owat.Support.Sorting (PVE v0.0.1)

        This module provides helper functions for sorting models.

        Requirements

        1. Every model must be equipped with a :sort_order field.
        2. After any operation that updates the database state, the sanitizing functions must be called.

        before(queryable, id)

        @spec before(Ecto.Queryable.t(), String.t() | nil) :: integer()

        Calculates the sort_order before a given entry by id.

        If an entry is given, the returned sort_order value is directly preceding the given entry. If nil is given, the returned sort_order value is last.
