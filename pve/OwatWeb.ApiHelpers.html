        OwatWeb.ApiHelpers (PVE v0.0.1)

        Common helper functions for OpenAPI response objects

        These functions generate default response-descriptor objects.

        Options

        Any function taking opts/0 may take these options:

        • :except to exclude specific response codes by their atom (meaning :not_found for 404 status code)
